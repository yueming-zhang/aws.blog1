## Part 1: Building the Performance Testing Harness

### Why Accurate Measurement is Harder Than It Looks

Before we write any code, it is worth understanding why measuring AgentCore performance accurately is non-trivial. There are several ways to get misleading data without realizing it:

**Problem 1: Session ID contamination**
If you accidentally reuse a session ID during a cold start test, you are measuring a warm invocation and calling it a cold start. Your cold start numbers will look unrealistically good.

**Problem 2: Warm-up contamination**
If you include the first invocation (which is always a cold start) in your warm invocation statistics, your p99 and mean latency figures will be significantly inflated.

**Problem 3: Insufficient sample size**
A single cold start measurement tells you almost nothing. Cold start times vary based on host load, ECR cache state, and network conditions. You need multiple iterations to get statistically meaningful data.

**Problem 4: Clock skew**
The timestamp on your client machine and the timestamp in CloudWatch Logs may differ by several seconds. If you use client-side timestamps to query CloudWatch Logs, you may miss spans or query the wrong time window.

**Problem 5: Mixed test types**
Running cold start and warm invocation tests in the same session produces data that is impossible to interpret correctly. The two test types must be completely separated.

Our testing harness is designed to eliminate all five of these problems.

#### AgentCore Client Wrapper

```python
import time
import logging
import boto3
import requests
from datetime import datetime
from typing import Optional
from dataclasses import dataclass, field

logger = logging.getLogger(__name__)


@dataclass
class InvocationResult:
    """
    Complete result from a single AgentCore invocation.
    Captures timing, response data, and metadata.
    """
    session_id: str
    start_time: datetime
    end_time: datetime
    duration_ms: float
    is_cold_start: bool
    success: bool
    response_payload: Optional[dict] = None
    error_message: Optional[str] = None
    request_id: Optional[str] = None


class AgentCoreClient:
    """
    Client wrapper for AWS Bedrock AgentCore API.
    
    Handles authentication, request formatting,
    and response parsing for MCP invocations.
    """
    
    def __init__(
        self,
        runtime_id: str,
        region: str = "us-east-1",
        endpoint_url: Optional[str] = None
    ):
        self.runtime_id = runtime_id
        self.region = region
        self.endpoint_url = (
            endpoint_url or
            f"https://bedrock-agentcore.{region}.amazonaws.com"
        )
        
        # Get credentials for request signing
        self.session = boto3.Session()
        self.credentials = self.session.get_credentials()
        
    def invoke(
        self,
        payload: dict,
        session_id: str,
        timeout_seconds: int = 300
    ) -> InvocationResult:
        """
        Invoke AgentCore runtime with timing measurement.
        
        Args:
            payload: Request payload dict
            session_id: Session ID for VM routing
            timeout_seconds: Request timeout
            
        Returns:
            InvocationResult with timing and response data
        """
        url = (
            f"{self.endpoint_url}"
            f"/runtimes/{self.runtime_id}/mcp"
        )
        
        headers = {
            "Content-Type": "application/json",
            "X-Session-Id": session_id,
        }
        
        # Sign the request with AWS SigV4
        headers = self._sign_request(
            method="POST",
            url=url,
            headers=headers,
            payload=payload
        )
        
        start_time = datetime.now()
        
        try:
            response = requests.post(
                url=url,
                json=payload,
                headers=headers,
                timeout=timeout_seconds
            )
            
            end_time = datetime.now()
            duration_ms = (
                end_time - start_time
            ).total_seconds() * 1000
            
            response.raise_for_status()
            
            return InvocationResult(
                session_id=session_id,
                start_time=start_time,
                end_time=end_time,
                duration_ms=duration_ms,
                is_cold_start=False,  # Caller sets this
                success=True,
                response_payload=response.json(),
                request_id=response.headers.get(
                    "X-Request-Id"
                )
            )
            
        except requests.exceptions.Timeout:
            end_time = datetime.now()
            duration_ms = (
                end_time - start_time
            ).total_seconds() * 1000
            
            logger.error(
                f"Request timed out after "
                f"{timeout_seconds}s for session "
                f"{session_id}"
            )
            
            return InvocationResult(
                session_id=session_id,
                start_time=start_time,
                end_time=end_time,
                duration_ms=duration_ms,
                is_cold_start=False,
                success=False,
                error_message=f"Timeout after {timeout_seconds}s"
            )
            
        except Exception as e:
            end_time = datetime.now()
            duration_ms = (
                end_time - start_time
            ).total_seconds() * 1000
            
            logger.error(
                f"Invocation failed for session "
                f"{session_id}: {e}"
            )
            
            return InvocationResult(
                session_id=session_id,
                start_time=start_time,
                end_time=end_time,
                duration_ms=duration_ms,
                is_cold_start=False,
                success=False,
                error_message=str(e)
            )
    
    def _sign_request(
        self,
        method: str,
        url: str,
        headers: dict,
        payload: dict
    ) -> dict:
        """Sign request with AWS SigV4 credentials."""
        # Implementation depends on your auth setup
        # This is a placeholder for the signing logic
        from botocore.auth import SigV4Auth
        from botocore.awsrequest import AWSRequest
        import json
        
        request = AWSRequest(
            method=method,
            url=url,
            data=json.dumps(payload),
            headers=headers
        )
        
        SigV4Auth(
            self.credentials,
            "bedrock",
            self.region
        ).add_auth(request)
        
        return dict(request.headers)
```

### Cold Start Testing

The cold start test is the most critical measurement to get right. The key invariant is simple but must never be violated: **every iteration must use a brand new session ID that AgentCore has never seen before**.

```python
import statistics
from dataclasses import dataclass, field
from typing import List, Dict


@dataclass
class ColdStartTestResult:
    """
    Complete results from a cold start test run.
    Contains per-iteration data and aggregate statistics.
    """
    runtime_id: str
    iterations: int
    successful_iterations: int
    failed_iterations: int
    
    # Aggregate timing statistics (milliseconds)
    total_mean_ms: float
    total_std_dev_ms: float
    total_min_ms: float
    total_max_ms: float
    
    # Phase-level statistics (milliseconds)
    vm_init_mean_ms: float
    runtime_startup_mean_ms: float
    first_request_mean_ms: float
    
    # Raw per-iteration data
    raw_durations_ms: List[float] = field(
        default_factory=list
    )
    
    # Test metadata
    test_start_time: Optional[datetime] = None
    test_end_time: Optional[datetime] = None


class ColdStartTester:
    """
    Measures AgentCore cold start performance.
    
    Guarantees measurement accuracy by:
    1. Using a NEW session ID for every iteration
    2. Terminating each session after measurement
    3. Running multiple iterations for statistical confidence
    4. Extracting phase-level timing from OTel spans
    """
    
    # Pause between iterations to avoid resource contention
    INTER_ITERATION_PAUSE_SECONDS = 3
    
    def __init__(
        self,
        client: AgentCoreClient,
        session_manager,
        trace_parser,
        runtime_id: str
    ):
        self.client = client
        self.session_manager = session_manager
        self.trace_parser = trace_parser
        self.runtime_id = runtime_id
        
    def run(
        self,
        iterations: int = 10,
        test_payload: Optional[dict] = None
    ) -> ColdStartTestResult:
        """
        Run cold start performance test.
        
        Args:
            iterations: Number of cold start measurements.
                       Minimum 10 recommended for statistical
                       confidence.
            test_payload: Payload to send with each request.
                         Defaults to a standard test query.
                         
        Returns:
            ColdStartTestResult with full statistics
        """
        if test_payload is None:
            test_payload = {
                "query": "Performance test — cold start measurement"
            }
        
        logger.info(
            f"Starting cold start test: "
            f"{iterations} iterations for runtime "
            f"{self.runtime_id}"
        )
        
        test_start_time = datetime.now()
        raw_durations = []
        phase_timings = {
            "vm_init": [],
            "runtime_startup": [],
            "first_request": []
        }
        failed_iterations = 0
        
        for i in range(iterations):
            logger.info(
                f"Cold start iteration "
                f"{i + 1}/{iterations}"
            )
            
            # CRITICAL: New session ID for every iteration
            # This is what guarantees a cold start
            session_id = self.session_manager.create_session(
                self.runtime_id
            )
            
            try:
                # Make the request and measure wall-clock time
                result = self.client.invoke(
                    payload=test_payload,
                    session_id=session_id,
                    timeout_seconds=300  # 5 min for cold start
                )
                
                if result.success:
                    raw_durations.append(result.duration_ms)
                    
                    logger.info(
                        f"Iteration {i + 1} completed: "
                        f"{result.duration_ms:.0f}ms"
                    )
                    
                    # Extract phase timings from OTel spans
                    # Allow time for spans to appear in logs
                    time.sleep(5)
                    
                    phases = self.trace_parser.extract_phases(
                        session_id=session_id,
                        invocation_start=result.start_time,
                        total_duration_ms=result.duration_ms
                    )
                    
                    if phases:
                        phase_timings["vm_init"].append(
                            phases.get("vm_init_ms", 0)
                        )
                        phase_timings["runtime_startup"].append(
                            phases.get("runtime_startup_ms", 0)
                        )
                        phase_timings["first_request"].append(
                            phases.get("first_request_ms", 0)
                        )
                else:
                    logger.warning(
                        f"Iteration {i + 1} failed: "
                        f"{result.error_message}"
                    )
                    failed_iterations += 1
                    
            except Exception as e:
                logger.error(
                    f"Iteration {i + 1} raised exception: {e}"
                )
                failed_iterations += 1
                
            finally:
                # Always terminate session after measurement
                # This ensures clean state for next iteration
                self.session_manager.terminate_session(
                    session_id
                )
                
                # Pause between iterations
                if i < iterations - 1:
                    time.sleep(
                        self.INTER_ITERATION_PAUSE_SECONDS
                    )
        
        test_end_time = datetime.now()
        
        # Calculate aggregate statistics
        if not raw_durations:
            raise RuntimeError(
                "All cold start iterations failed. "
                "Check AgentCore connectivity and credentials."
            )
        
        return ColdStartTestResult(
            runtime_id=self.runtime_id,
            iterations=iterations,
            successful_iterations=len(raw_durations),
            failed_iterations=failed_iterations,
            total_mean_ms=statistics.mean(raw_durations),
            total_std_dev_ms=(
                statistics.stdev(raw_durations)
                if len(raw_durations) > 1
                else 0
            ),
            total_min_ms=min(raw_durations),
            total_max_ms=max(raw_durations),
            vm_init_mean_ms=(
                statistics.mean(phase_timings["vm_init"])
                if phase_timings["vm_init"]
                else 0
            ),
            runtime_startup_mean_ms=(
                statistics.mean(
                    phase_timings["runtime_startup"]
                )
                if phase_timings["runtime_startup"]
                else 0
            ),
            first_request_mean_ms=(
                statistics.mean(
                    phase_timings["first_request"]
                )
                if phase_timings["first_request"]
                else 0
            ),
            raw_durations_ms=raw_durations,
            test_start_time=test_start_time,
            test_end_time=test_end_time
        )
```

### Warm Invocation Testing

The warm invocation test has the opposite requirement from the cold start test: **every iteration must use the exact same session ID**. The first invocation (which is always a cold start) must be discarded before measurements begin.

```python
@dataclass
class WarmInvocationTestResult:
    """
    Complete results from a warm invocation test run.
    Contains percentile statistics and raw latency data.
    """
    runtime_id: str
    iterations: int
    successful_iterations: int
    failed_iterations: int
    
    # Percentile statistics (milliseconds)
    p50_ms: float
    p90_ms: float
    p95_ms: float
    p99_ms: float
    mean_ms: float
    std_dev_ms: float
    min_ms: float
    max_ms: float
    
    # Raw per-iteration latencies
    raw_latencies_ms: List[float] = field(
        default_factory=list
    )
    
    # Test metadata
    session_id: Optional[str] = None
    test_start_time: Optional[datetime] = None
    test_end_time: Optional[datetime] = None


class WarmInvocationTester:
    """
    Measures AgentCore warm invocation performance.
    
    Guarantees measurement accuracy by:
    1. Creating ONE session and reusing it for all iterations
    2. Discarding the first (cold start) invocation
    3. Running 100+ iterations for p99 accuracy
    4. Keeping session alive with record_invocation()
    5. Checking session health before each request
    """
    
    def __init__(
        self,
        client: AgentCoreClient,
        session_manager,
        runtime_id: str
    ):
        self.client = client
        self.session_manager = session_manager
        self.runtime_id = runtime_id
        
    def run(
        self,
        iterations: int = 100,
        test_payload: Optional[dict] = None
    ) -> WarmInvocationTestResult:
        """
        Run warm invocation performance test.
        
        Args:
            iterations: Number of warm invocation measurements.
                       Minimum 100 recommended for accurate p99.
            test_payload: Payload to send with each request.
            
        Returns:
            WarmInvocationTestResult with percentile statistics
        """
        if test_payload is None:
            test_payload = {
                "query": "Performance test — warm invocation"
            }
        
        logger.info(
            f"Starting warm invocation test: "
            f"{iterations} iterations for runtime "
            f"{self.runtime_id}"
        )
        
        # Step 1: Create ONE session for all iterations
        session_id = self.session_manager.create_session(
            self.runtime_id
        )
        
        logger.info(
            f"Created session {session_id} — "
            f"running warm-up invocation"
        )
        
        # Step 2: Warm-up invocation
        # This triggers the cold start — we discard this timing
        warmup_result = self.client.invoke(
            payload={"query": "Warm-up — discard this timing"},
            session_id=session_id,
            timeout_seconds=300
        )
        
        if not warmup_result.success:
            raise RuntimeError(
                f"Warm-up invocation failed: "
                f"{warmup_result.error_message}. "
                f"Cannot proceed with warm invocation test."
            )
        
        self.session_manager.record_invocation(session_id)
        
        logger.info(
            f"Warm-up complete "
            f"({warmup_result.duration_ms:.0f}ms — discarded). "
            f"Starting measurement loop."
        )
        
        # Step 3: Measurement loop
        test_start_time = datetime.now()
        raw_latencies = []
        failed_iterations = 0
        
        for i in range(iterations):
            # Verify session is still warm before each request
            if not self.session_manager.is_session_warm(
                session_id
            ):
                logger.warning(
                    f"Session expired at iteration {i + 1}. "
                    f"Creating new session — "
                    f"this iteration will be excluded."
                )
                session_id = self.session_manager.create_session(
                    self.runtime_id
                )
                # Run a new warm-up and skip this iteration
                self.client.invoke(
                    payload={"query": "Re-warm after expiry"},
                    session_id=session_id,
                    timeout_seconds=300
                )
                self.session_manager.record_invocation(
                    session_id
                )
                continue
            
            # Make the warm invocation request
            result = self.client.invoke(
                payload=test_payload,
                session_id=session_id  # SAME session every time
            )
            
            if result.success:
                raw_latencies.append(result.duration_ms)
                
                # Keep session alive
                self.session_manager.record_invocation(
                    session_id
                )
                
                if (i + 1) % 10 == 0:
                    logger.info(
                        f"Progress: {i + 1}/{iterations} "
                        f"iterations complete. "
                        f"Last: {result.duration_ms:.0f}ms"
                    )
            else:
                logger.warning(
                    f"Iteration {i + 1} failed: "
                    f"{result.error_message}"
                )
                failed_iterations += 1
        
        test_end_time = datetime.now()
        
        if not raw_latencies:
            raise RuntimeError(
                "All warm invocation iterations failed."
            )
        
        # Step 4: Calculate percentile statistics
        return self._calculate_statistics(
            raw_latencies=raw_latencies,
            iterations=iterations,
            failed_iterations=failed_iterations,
            session_id=session_id,
            test_start_time=test_start_time,
            test_end_time=test_end_time
        )
    
    def _calculate_statistics(
        self,
        raw_latencies: List[float],
        iterations: int,
        failed_iterations: int,
        session_id: str,
        test_start_time: datetime,
        test_end_time: datetime
    ) -> WarmInvocationTestResult:
        """Calculate percentile statistics from raw latencies."""
        
        sorted_latencies = sorted(raw_latencies)
        n = len(sorted_latencies)
        
        def percentile(p: float) -> float:
            """Calculate the p-th percentile."""
            index = int(n * p / 100)
            # Clamp to valid range
            index = max(0, min(index, n - 1))
            return sorted_latencies[index]
        
        return WarmInvocationTestResult(
            runtime_id=self.client.runtime_id,
            iterations=iterations,
            successful_iterations=n,
            failed_iterations=failed_iterations,
            p50_ms=percentile(50),
            p90_ms=percentile(90),
            p95_ms=percentile(95),
            p99_ms=percentile(99),
            mean_ms=statistics.mean(raw_latencies),
            std_dev_ms=(
                statistics.stdev(raw_latencies)
                if n > 1 else 0
            ),
            min_ms=min(raw_latencies),
            max_ms=max(raw_latencies),
            raw_latencies_ms=raw_latencies,
            session_id=session_id,
            test_start_time=test_start_time,
            test_end_time=test_end_time
        )
```

## Part 2: Extracting Phase Timings from OpenTelemetry Spans

One of the most powerful capabilities in the AgentCore observability stack is the ability to break down cold start time into its three constituent phases using OpenTelemetry span data. This tells you not just how long the cold start took in total, but exactly where within those 40 seconds the time was spent.

### How OTel Spans Flow Through the Stack

When AgentCore processes a request, it emits OpenTelemetry spans at each phase boundary. These spans are written to the `aws/spans` CloudWatch Logs log group, tagged with the session ID that generated them. By querying this log group with the session ID as a filter, you can reconstruct the timing of each phase for any given invocation.

The three spans we care about are:

| Span Name | Phase | What It Measures |
|-----------|-------|-----------------|
| `firecracker_vm_init` | Phase 1 | VM boot through image pull |
| `runtime_startup` | Phase 2 | Python init through graph compile |
| `request_processing` | Phase 3 | Payload handling through response |

### The Trace Parser

```python
import json
from datetime import datetime, timedelta
from typing import Optional, Dict
import boto3


class TraceParser:
    """
    Extracts phase-level timing from OTel span data
    stored in CloudWatch Logs.
    
    Handles clock-skew between client and CloudWatch
    by using a configurable time window around the
    known invocation start time.
    """
    
    # Log group where AgentCore writes OTel spans
    LOG_GROUP = "aws/spans"
    
    # How far back to search for spans
    # (accounts for clock skew and log ingestion delay)
    SEARCH_WINDOW_SECONDS = 120
    
    # How long to wait for spans to appear in logs
    LOG_INGESTION_DELAY_SECONDS = 10
    
    def __init__(self, region: str = "us-east-1"):
        self.logs_client = boto3.client(
            "logs",
            region_name=region
        )
        
    def extract_phases(
        self,
        session_id: str,
        invocation_start: datetime,
        total_duration_ms: float
    ) -> Optional[Dict[str, float]]:
        """
        Extract phase-level timing for a specific invocation.
        
        Args:
            session_id: Session ID to search for in logs
            invocation_start: When the invocation started
                             (client-side timestamp)
            total_duration_ms: Total invocation duration
                               (used to calculate vm_init
                               when span is not available)
                               
        Returns:
            Dict with vm_init_ms, runtime_startup_ms,
            first_request_ms, or None if spans not found
        """
        logger.info(
            f"Extracting phase timings for "
            f"session {session_id}"
        )
        
        # Wait for log ingestion
        time.sleep(self.LOG_INGESTION_DELAY_SECONDS)
        
        # Define search window with clock-skew buffer
        start_ms = int(
            (
                invocation_start -
                timedelta(seconds=30)  # Clock skew buffer
            ).timestamp() * 1000
        )
        end_ms = int(
            (
                invocation_start +
                timedelta(
                    milliseconds=total_duration_ms,
                    seconds=30  # Buffer for late spans
                )
            ).timestamp() * 1000
        )
        
        # Query CloudWatch Logs for spans
        spans = self._query_spans(
            session_id=session_id,
            start_ms=start_ms,
            end_ms=end_ms
        )
        
        if not spans:
            logger.warning(
                f"No spans found for session {session_id}. "
                f"Phase breakdown will be estimated."
            )
            return self._estimate_phases(total_duration_ms)
        
        return self._parse_phase_timings(
            spans=spans,
            total_duration_ms=total_duration_ms
        )
    
    def _query_spans(
        self,
        session_id: str,
        start_ms: int,
        end_ms: int
    ) -> list:
        """Query CloudWatch Logs for OTel spans."""
        
        query = f"""
            fields @timestamp, @message
            | filter sessionId = "{session_id}"
            | filter spanName in [
                "firecracker_vm_init",
                "runtime_startup",
                "request_processing"
              ]
            | sort @timestamp asc
        """
        
        try:
            # Start the query
            response = self.logs_client.start_query(
                logGroupName=self.LOG_GROUP,
                startTime=start_ms,
                endTime=end_ms,
                queryString=query
            )
            
            query_id = response["queryId"]
            
            # Poll until query completes
            results = self._wait_for_query(query_id)
            
            return results
            
        except Exception as e:
            logger.error(
                f"CloudWatch Logs query failed: {e}"
            )
            return []
    
    def _wait_for_query(
        self,
        query_id: str,
        max_wait_seconds: int = 30
    ) -> list:
        """Poll CloudWatch Logs Insights until query completes."""
        
        start = time.time()
        
        while time.time() - start < max_wait_seconds:
            response = self.logs_client.get_query_results(
                queryId=query_id
            )
            
            status = response["status"]
            
            if status == "Complete":
                return response.get("results", [])
            elif status in ["Failed", "Cancelled"]:
                logger.error(
                    f"Query {query_id} ended with "
                    f"status: {status}"
                )
                return []
            
            # Query still running — wait and retry
            time.sleep(2)
        
        logger.warning(
            f"Query {query_id} timed out after "
            f"{max_wait_seconds}s"
        )
        return []
    
    def _parse_phase_timings(
        self,
        spans: list,
        total_duration_ms: float
    ) -> Dict[str, float]:
        """
        Parse phase timings from raw span data.
        
        For phases where spans are available, use span
        duration directly. For platform-managed phases
        (vm_init) where spans may not be available,
        calculate as remainder.
        """
        phase_durations = {}
        
        for result in spans:
            # Parse the span message
            message_field = next(
                (
                    field["value"]
                    for field in result
                    if field["field"] == "@message"
                ),
                None
            )
            
            if not message_field:
                continue
                
            try:
                span_data = json.loads(message_field)
                span_name = span_data.get("spanName", "")
                duration_ms = span_data.get("durationMs", 0)
                
                if span_name == "runtime_startup":
                    phase_durations["runtime_startup_ms"] = (
                        duration_ms
                    )
                elif span_name == "request_processing":
                    phase_durations["first_request_ms"] = (
                        duration_ms
                    )
                    
            except json.JSONDecodeError:
                logger.warning(
                    f"Could not parse span message: "
                    f"{message_field[:100]}"
                )
        
        # Calculate vm_init as remainder
        # (platform-managed, not directly instrumented)
        runtime_ms = phase_durations.get(
            "runtime_startup_ms", 0
        )
        request_ms = phase_durations.get(
            "first_request_ms", 0
        )
        
        phase_durations["vm_init_ms"] = max(
            0,
            total_duration_ms - runtime_ms - request_ms
        )
        
        return phase_durations
    
    def _estimate_phases(
        self,
        total_duration_ms: float
    ) -> Dict[str, float]:
        """
        Estimate phase breakdown when spans are unavailable.
        Uses typical percentage distribution as fallback.
        """
        return {
            "vm_init_ms": total_duration_ms * 0.40,
            "runtime_startup_ms": total_duration_ms * 0.30,
            "first_request_ms": total_duration_ms * 0.30
        }
```

## Part 5: Regression Detection

One of the most valuable capabilities in a mature performance testing setup is automatic regression detection. Rather than manually comparing numbers after every deployment, you want the system to tell you when performance has degraded beyond an acceptable threshold.

### Baseline Management

```python
import json
import boto3
from datetime import datetime


class BaselineManager:
    """
    Manages performance baselines stored in S3.
    
    Baselines are versioned JSON files that capture
    the expected performance characteristics of a
    runtime at a given point in time.
    
    When a new test run completes, its results are
    compared against the stored baseline to detect
    regressions automatically.
    """
    
    def __init__(
        self,
        bucket_name: str,
        region: str = "us-east-1"
    ):
        self.bucket_name = bucket_name
        self.s3 = boto3.client("s3", region_name=region)
        
    def save_baseline(
        self,
        runtime_id: str,
        environment: str,
        cold_start_result: ColdStartTestResult,
        warm_result: WarmInvocationTestResult
    ):
        """
        Save current test results as the new baseline.
        
        Call this after a known-good deployment to
        establish the reference point for future
        regression comparisons.
        """
        baseline = {
            "runtime_id": runtime_id,
            "environment": environment,
            "created_at": datetime.utcnow().isoformat(),
            "cold_start": {
                "mean_ms": cold_start_result.total_mean_ms,
                "std_dev_ms": cold_start_result.total_std_dev_ms,
                "vm_init_mean_ms": (
                    cold_start_result.vm_init_mean_ms
                ),
                "runtime_startup_mean_ms": (
                    cold_start_result.runtime_startup_mean_ms
                ),
                "first_request_mean_ms": (
                    cold_start_result.first_request_mean_ms
                )
            },
            "warm_invocation": {
                "p50_ms": warm_result.p50_ms,
                "p90_ms": warm_result.p90_ms,
                "p95_ms": warm_result.p95_ms,
                "p99_ms": warm_result.p99_ms,
                "mean_ms": warm_result.mean_ms,
                "std_dev_ms": warm_result.std_dev_ms
            }
        }
        
        key = (
            f"baselines/{runtime_id}/"
            f"{environment}/latest.json"
        )
        
        # Also save a timestamped version for history
        history_key = (
            f"baselines/{runtime_id}/{environment}/"
            f"{datetime.utcnow().strftime('%Y%m%d_%H%M%S')}"
            f".json"
        )
        
        baseline_json = json.dumps(baseline, indent=2)
        
        for s3_key in [key, history_key]:
            self.s3.put_object(
                Bucket=self.bucket_name,
                Key=s3_key,
                Body=baseline_json,
                ContentType="application/json"
            )
        
        logger.info(
            f"Saved baseline for {runtime_id} "
            f"({environment}) to s3://{self.bucket_name}/{key}"
        )
        
    def load_baseline(
        self,
        runtime_id: str,
        environment: str
    ) -> Optional[dict]:
        """Load the current baseline for a runtime."""
        
        key = (
            f"baselines/{runtime_id}/"
            f"{environment}/latest.json"
        )
        
        try:
            response = self.s3.get_object(
                Bucket=self.bucket_name,
                Key=key
            )
            return json.loads(
                response["Body"].read().decode("utf-8")
            )
        except self.s3.exceptions.NoSuchKey:
            logger.warning(
                f"No baseline found for {runtime_id} "
                f"({environment}). "
                f"Run with --save-baseline to create one."
            )
            return None
        except Exception as e:
            logger.error(f"Failed to load baseline: {e}")
            return None


class RegressionDetector:
    """
    Detects performance regressions by comparing
    current test results against stored baselines.
    
    A regression is defined as any metric that has
    degraded by more than the configured threshold
    percentage compared to the baseline.
    """
    
    # Default regression threshold: 20% degradation
    DEFAULT_THRESHOLD_PCT = 20.0
    
    def __init__(
        self,
        baseline_manager: BaselineManager,
        threshold_pct: float = DEFAULT_THRESHOLD_PCT
    ):
        self.baseline_manager = baseline_manager
        self.threshold_pct = threshold_pct
        
    def check_for_regressions(
        self,
        runtime_id: str,
        environment: str,
        cold_start_result: ColdStartTestResult,
        warm_result: WarmInvocationTestResult
    ) -> dict:
        """
        Compare current results against baseline
        and identify any regressions.
        
        Returns:
            Dict with 'regressions' list and
            'has_regression' boolean
        """
        baseline = self.baseline_manager.load_baseline(
            runtime_id=runtime_id,
            environment=environment
        )
        
        if baseline is None:
            return {
                "has_regression": False,
                "regressions": [],
                "message": "No baseline available for comparison"
            }
        
        regressions = []
        
        # Check cold start metrics
        cold_checks = [
            (
                "ColdStart_Mean",
                cold_start_result.total_mean_ms,
                baseline["cold_start"]["mean_ms"]
            ),
            (
                "ColdStart_VMInit",
                cold_start_result.vm_init_mean_ms,
                baseline["cold_start"]["vm_init_mean_ms"]
            ),
            (
                "ColdStart_RuntimeStartup",
                cold_start_result.runtime_startup_mean_ms,
                baseline["cold_start"]["runtime_startup_mean_ms"]
            ),
        ]
        
        # Check warm invocation metrics
        warm_checks = [
            (
                "WarmInvocation_P50",
                warm_result.p50_ms,
                baseline["warm_invocation"]["p50_ms"]
            ),
            (
                "WarmInvocation_P90",
                warm_result.p90_ms,
                baseline["warm_invocation"]["p90_ms"]
            ),
            (
                "WarmInvocation_P95",
                warm_result.p95_ms,
                baseline["warm_invocation"]["p95_ms"]
            ),
            (
                "WarmInvocation_P99",
                warm_result.p99_ms,
                baseline["warm_invocation"]["p99_ms"]
            ),
            (
                "WarmInvocation_Mean",
                warm_result.mean_ms,
                baseline["warm_invocation"]["mean_ms"]
            ),
        ]
        
        all_checks = cold_checks + warm_checks
        
        for metric_name, current_value, baseline_value in (
            all_checks
        ):
            if baseline_value == 0:
                continue
                
            degradation_pct = (
                (current_value - baseline_value) /
                baseline_value * 100
            )
            
            if degradation_pct > self.threshold_pct:
                severity = (
                    "CRITICAL"
                    if degradation_pct > 50
                    else "WARNING"
                )
                
                regression = {
                    "metric": metric_name,
                    "baseline_ms": round(baseline_value, 1),
                    "current_ms": round(current_value, 1),
                    "degradation_pct": round(
                        degradation_pct, 1
                    ),
                    "severity": severity
                }
                
                regressions.append(regression)
                
                logger.warning(
                    f"REGRESSION DETECTED: {metric_name} "
                    f"degraded by {degradation_pct:.1f}% "
                    f"({baseline_value:.0f}ms → "
                    f"{current_value:.0f}ms) [{severity}]"
                )
        
        return {
            "has_regression": len(regressions) > 0,
            "regressions": regressions,
            "baseline_created_at": baseline.get("created_at"),
            "threshold_pct": self.threshold_pct
        }
```

                region=self.region
            )
            self.regression_detector = RegressionDetector(
                baseline_manager=self.baseline_manager
            )
        else:
            self.baseline_manager = None
            self.regression_detector = None
            
        if self.sns_topic_arn:
            self.alarm_manager = AlarmManager(
                region=self.region
            )
        else:
            self.alarm_manager = None
    
    def run(
        self,
        test_type: str = "both",
        cold_iterations: int = 10,
        warm_iterations: int = 100,
        save_baseline: bool = False,
        setup_alarms: bool = False
    ) -> dict:
        """
        Run the complete performance test suite.
        
        Args:
            test_type: "cold", "warm", or "both"
            cold_iterations: Iterations for cold start test
            warm_iterations: Iterations for warm invocation test
            save_baseline: Save results as new baseline
            setup_alarms: Create CloudWatch alarms
            
        Returns:
            Complete results dict with all metrics
        """
        logger.info(
            f"Starting AgentCore performance test suite\n"
            f"  Runtime:     {self.runtime_id}\n"
            f"  Environment: {self.environment}\n"
            f"  Test type:   {test_type}\n"
            f"  Cold iters:  {cold_iterations}\n"
            f"  Warm iters:  {warm_iterations}"
        )
        
        results = {
            "runtime_id": self.runtime_id,
            "environment": self.environment,
            "test_type": test_type,
            "run_timestamp": datetime.utcnow().isoformat(),
            "cold_start": None,
            "warm_invocation": None,
            "regression_check": None
        }
        
        cold_result = None
        warm_result = None
        
        # Run cold start test
        if test_type in ["cold", "both"]:
            logger.info("=" * 50)
            logger.info("COLD START TEST")
            logger.info("=" * 50)
            
            cold_result = self.cold_tester.run(
                iterations=cold_iterations
            )
            
            self.metrics_publisher.publish_cold_start_results(
                result=cold_result,
                environment=self.environment
            )
            
            results["cold_start"] = {
                "mean_ms": round(cold_result.total_mean_ms, 1),
                "std_dev_ms": round(
                    cold_result.total_std_dev_ms, 1
                ),
                "min_ms": round(cold_result.total_min_ms, 1),
                "max_ms": round(cold_result.total_max_ms, 1),
                "vm_init_mean_ms": round(
                    cold_result.vm_init_mean_ms, 1
                ),
                "runtime_startup_mean_ms": round(
                    cold_result.runtime_startup_mean_ms, 1
                ),
                "first_request_mean_ms": round(
                    cold_result.first_request_mean_ms, 1
                ),
                "successful_iterations": (
                    cold_result.successful_iterations
                ),
                "failed_iterations": (
                    cold_result.failed_iterations
                )
            }
            
            self._print_cold_start_summary(cold_result)
        
        # Run warm invocation test
        if test_type in ["warm", "both"]:
            logger.info("=" * 50)
            logger.info("WARM INVOCATION TEST")
            logger.info("=" * 50)
            
            warm_result = self.warm_tester.run(
                iterations=warm_iterations
            )
            
            self.metrics_publisher.publish_warm_invocation_results(
                result=warm_result,
                environment=self.environment
            )
            
            results["warm_invocation"] = {
                "p50_ms": round(warm_result.p50_ms, 1),
                "p90_ms": round(warm_result.p90_ms, 1),
                "p95_ms": round(warm_result.p95_ms, 1),
                "p99_ms": round(warm_result.p99_ms, 1),
                "mean_ms": round(warm_result.mean_ms, 1),
                "std_dev_ms": round(warm_result.std_dev_ms, 1),
                "min_ms": round(warm_result.min_ms, 1),
                "max_ms": round(warm_result.max_ms, 1),
                "successful_iterations": (
                    warm_result.successful_iterations
                ),
                "failed_iterations": (
                    warm_result.failed_iterations
                )
            }
            
            self._print_warm_invocation_summary(warm_result)
        
        # Regression detection
        if (
            self.regression_detector and
            cold_result and
            warm_result
        ):
            logger.info("=" * 50)
            logger.info("REGRESSION CHECK")
            logger.info("=" * 50)
            
            regression_check = (
                self.regression_detector.check_for_regressions(
                    runtime_id=self.runtime_id,
                    environment=self.environment,
                    cold_start_result=cold_result,
                    warm_result=warm_result
                )
            )
            
            results["regression_check"] = regression_check
            
            if regression_check["has_regression"]:
                logger.warning(
                    f"REGRESSIONS DETECTED: "
                    f"{len(regression_check['regressions'])} "
                    f"metric(s) degraded beyond threshold"
                )
                for r in regression_check["regressions"]:
                    logger.warning(
                        f"  [{r['severity']}] {r['metric']}: "
                        f"{r['baseline_ms']}ms → "
                        f"{r['current_ms']}ms "
                        f"(+{r['degradation_pct']}%)"
                    )
            else:
                logger.info(
                    "No regressions detected. "
                    "Performance within baseline thresholds."
                )
        
        # Save as new baseline if requested
        if (
            save_baseline and
            self.baseline_manager and
            cold_result and
            warm_result
        ):
            self.baseline_manager.save_baseline(
                runtime_id=self.runtime_id,
                environment=self.environment,
                cold_start_result=cold_result,
                warm_result=warm_result
            )
            logger.info("Results saved as new baseline.")
        
        # Set up CloudWatch alarms if requested
        if setup_alarms and self.alarm_manager:
            self.alarm_manager.create_sla_alarms(
                runtime_id=self.runtime_id,
                environment=self.environment,
                sns_topic_arn=self.sns_topic_arn
            )
        
        return results
    
    def _print_cold_start_summary(
        self,
        result: ColdStartTestResult
    ):
        """Print formatted cold start results."""
        logger.info(
            f"\n{'=' * 50}\n"
            f"COLD START RESULTS\n"
            f"{'=' * 50}\n"
            f"  Total Mean:        {result.total_mean_ms:.0f}ms\n"
            f"  Total Std Dev:     "
            f"{result.total_std_dev_ms:.0f}ms\n"
            f"  Total Min:         {result.total_min_ms:.0f}ms\n"
            f"  Total Max:         {result.total_max_ms:.0f}ms\n"
            f"  ---\n"
            f"  VM Init Mean:      "
            f"{result.vm_init_mean_ms:.0f}ms\n"
            f"  Runtime Startup:   "
            f"{result.runtime_startup_mean_ms:.0f}ms\n"
            f"  First Request:     "
            f"{result.first_request_mean_ms:.0f}ms\n"
            f"  ---\n"
            f"  Successful:        "
            f"{result.successful_iterations}/"
            f"{result.iterations}\n"
            f"{'=' * 50}"
        )
    
    def _print_warm_invocation_summary(
        self,
        result: WarmInvocationTestResult
    ):
        """Print formatted warm invocation results."""
        logger.info(
            f"\n{'=' * 50}\n"
            f"WARM INVOCATION RESULTS\n"
            f"{'=' * 50}\n"
            f"  P50:               {result.p50_ms:.0f}ms\n"
            f"  P90:               {result.p90_ms:.0f}ms\n"
            f"  P95:               {result.p95_ms:.0f}ms\n"
            f"  P99:               {result.p99_ms:.0f}ms\n"
            f"  Mean:              {result.mean_ms:.0f}ms\n"
            f"  Std Dev:           {result.std_dev_ms:.0f}ms\n"
            f"  Min:               {result.min_ms:.0f}ms\n"
            f"  Max:               {result.max_ms:.0f}ms\n"
            f"  ---\n"
            f"  Successful:        "
            f"{result.successful_iterations}/"
            f"{result.iterations}\n"
            f"{'=' * 50}"
        )

## Part 7: Optimization Strategies with Measured Results

Theory is useful, but measured results are what get optimization work prioritized and approved. Here is a concrete optimization walkthrough with before-and-after data.

### Optimization 1: Docker Image Size Reduction

**Before:** Standard Python base image with all dependencies installed in a single layer.

```dockerfile
# Before: Single-stage build
FROM python:3.11
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY src/ .
CMD ["python", "agent.py"]
# Resulting image size: ~2.1 GB
```

**After:** Multi-stage build with slim base image and cleaned pip cache.

```dockerfile
# After: Multi-stage optimized build
FROM python:3.11 AS builder
WORKDIR /build
COPY requirements.txt .
RUN pip install --no-cache-dir \
    --target /build/packages \
    -r requirements.txt

FROM python:3.11-slim AS runtime
WORKDIR /app
COPY --from=builder /build/packages /app/packages
COPY src/ /app/src/
ENV PYTHONPATH=/app/packages
CMD ["python", "src/agent.py"]
# Resulting image size: ~420 MB
```

**Measured impact:**

| Metric | Before | After | Improvement |
|--------|--------|-------|-------------|
| Image size | 2.1 GB | 420 MB | -80% |
| VM Init (Phase 1) | 24,200ms | 9,800ms | -59% |
| Total Cold Start | 52,400ms | 31,600ms | -40% |

### Optimization 2: Lazy Import Loading

**Before:** All imports at module level.

```python
# Before: All imports at startup
import torch
import transformers
from langchain.embeddings import HuggingFaceEmbeddings
from langchain.vectorstores import FAISS
# Runtime startup: ~14,200ms
```

**After:** Heavy imports deferred to first use.

```python
# After: Lazy imports
def get_embeddings():
    from langchain.embeddings import HuggingFaceEmbeddings
    return HuggingFaceEmbeddings()

def get_vectorstore(embeddings):
    from langchain.vectorstores import FAISS
    return FAISS.load_local("index", embeddings)
# Runtime startup: ~6,800ms
```

**Measured impact:**

| Metric | Before | After | Improvement |
|--------|--------|-------|-------------|
| Runtime Startup | 14,200ms | 6,800ms | -52% |
| Total Cold Start | 52,400ms | 44,000ms | -16% |

### Combined Optimization Results

Applying both optimizations together:

| Metric | Baseline | Optimized | Total Improvement |
|--------|----------|-----------|------------------|
| VM Init | 24,200ms | 9,800ms | -59% |
| Runtime Startup | 14,200ms | 6,800ms | -52% |
| First Request | 14,000ms | 12,100ms | -14% |
| **Total Cold Start** | **52,400ms** | **28,700ms** | **-45%** |
| Warm P50 | 3,800ms | 3,200ms | -16% |
| Warm P99 | 8,200ms | 6,900ms | -16% |

A 45% reduction in cold start time — from 52 seconds to 29 seconds — achieved through two focused optimizations that took less than a day to implement.

## Summary and Key Takeaways

In this two-part series, we have built a complete understanding of AgentCore performance from the ground up.

**From Part 1**, you learned:
- Firecracker microVMs provide the secure, isolated execution environment for each AgentCore session
- Session IDs are routing keys that determine whether you get a cold start or a warm start
- Cold starts involve seven stages and typically take 35-55 seconds
- Warm starts skip all initialization and typically complete in 2-8 seconds
- The SessionManager pattern is essential for production session lifecycle management

**From Part 2**, you learned:
- Accurate measurement requires strict separation of cold start and warm invocation tests
- The three-phase timing model (VM Init, Runtime Startup, First Request) gives you actionable insight into where cold start time is spent
- CloudWatch custom metrics under the `AgentCore/PerformanceTesting` namespace provide the foundation for dashboards and alerting
- OTel span data in CloudWatch Logs enables phase-level timing extraction
- Regression detection against S3-stored baselines catches performance degradations automatically
- Docker image optimization and lazy imports can reduce cold start time by 40-50%

**The five most important things to take away:**

1. **Measure before you optimize.** Without a baseline, you cannot know whether your changes are helping or hurting.

2. **Separate your test types.** Cold start and warm invocation measurements must never be mixed. Use new session IDs for cold tests and reuse session IDs for warm tests.

3. **Use enough iterations.** Ten iterations minimum for cold start confidence. One hundred iterations minimum for accurate p99 warm invocation data.

4. **Set up alarms before you go to production.** Do not wait for users to tell you about performance problems. CloudWatch alarms with p99 thresholds will catch issues before they become incidents.

5. **Establish baselines after every known-good deployment.** Regression detection is only as good as the baseline it compares against. Keep your baselines current.
