#### **Section 5: Cold Start Deep Dive (~550 words)**

**What to Write:**

**5a — What Triggers a Cold Start:**
```
Trigger 1: No session ID provided
Trigger 2: New (never-used) session ID
Trigger 3: Session idle > 15 minutes
Trigger 4: Session explicitly terminated
```

**5b — The 7-Stage Instantiation Process:**
```
Stage 1: Authentication & Routing
  Owner: Platform
  What: Validate AWS SigV4/OAuth 2.0
  Time: Included in platform overhead

Stage 2: Resource Allocation  
  Owner: Platform
  What: Find available VM slot, reserve 2vCPU/8GB
  Time: Included in platform overhead

Stage 3: MicroVM Boot
  Owner: Platform
  What: Start Firecracker VM, initialize kernel
  Time: Included in platform overhead

Stage 4: Image Pull & Decompression ⚠️ BOTTLENECK
  Owner: Platform
  What: Pull Docker image from ECR
  Time: Biggest variable — depends on image size

Stage 5: Runtime Initialization
  Owner: Your Code
  What: Start Python 3.11 interpreter
  Time: ~2-3 seconds typical

Stage 6: Dependency Loading
  Owner: Your Code
  What: Import LangGraph, LangChain, Bedrock SDK
  Time: ~5-8 seconds typical

Stage 7: Entrypoint Execution
  Owner: Your Code
  What: Create agent, compile graph
  Time: ~3-5 seconds typical
```

**5c — The 3-Phase Timing Model:**
```
Phase 1: Firecracker VM Init
  = Stages 1-4 (Platform managed)
  Typical: 15-25 seconds (40% of cold start)
  
Phase 2: Runtime Startup  
  = Stages 5-7 (Your code)
  Typical: 10-15 seconds (30% of cold start)

Phase 3: First Request Processing
  = Request handling
  Typical: 10-15 seconds (30% of cold start)

Total Cold Start = Phase 1 + Phase 2 + Phase 3
Typical Total: 35-55 seconds
```

**Add Session Management Code:**
```python
import uuid
from datetime import datetime

class SessionManager:
    """
    Manages AgentCore session IDs to maximize 
    warm starts and minimize cold starts
    """
    def __init__(self):
        self.sessions = {}
        self.session_last_used = {}
        self.SESSION_TTL_SECONDS = 900  # 15 minutes
    
    def create_session(self, runtime_id: str) -> str:
        """
        Create new session — forces cold start
        Use for: Cold start testing, new user sessions
        """
        session_id = str(uuid.uuid4())
        
        if runtime_id not in self.sessions:
            self.sessions[runtime_id] = []
            
        self.sessions[runtime_id].append(session_id)
        self.session_last_used[session_id] = datetime.now()
        
        return session_id
    
    def get_warm_session(self, runtime_id: str) -> str:
        """
        Get existing warm session — avoids cold start
        Use for: Production requests, warm testing
        Returns None if no warm session available
        """
        sessions = self.sessions.get(runtime_id, [])
        
        for session_id in sessions:
            if self.is_session_warm(session_id):
                return session_id
                
        return None  # No warm session — caller must create new
    
    def is_session_warm(self, session_id: str) -> bool:
        """Check if session is within 15-min TTL"""
        if session_id not in self.session_last_used:
            return False
            
        last_used = self.session_last_used[session_id]
        age = (datetime.now() - last_used).total_seconds()
        
        return age < self.SESSION_TTL_SECONDS
    
    def record_invocation(self, session_id: str):
        """
        Update last-used timestamp
        Call this after every successful invocation
        to keep session alive
        """
        self.session_last_used[session_id] = datetime.now()
```

**4a — The Warm Measurement Pattern:**

```python
def run_warm_invocation_test(
    runtime_id: str,
    iterations: int = 100
) -> dict:
    """
    Accurate warm invocation measurement.
    Key: SAME session ID for ALL iterations.
    """
    session_manager = SessionManager()
    latencies = []
    
    # Step 1: Create ONE session
    session_id = session_manager.create_session(runtime_id)
    
    # Step 2: Warm-up invocation (DISCARD this timing)
    print("Running warm-up invocation (excluded from results)")
    agentcore_client.invoke(
        runtime_id=runtime_id,
        payload={"query": "Warm-up query"},
        session_id=session_id
    )
    session_manager.record_invocation(session_id)
    
    # Step 3: Measure loop — REUSE same session_id
    print(f"Running {iterations} warm invocations...")
    
    for i in range(iterations):
        start_time = datetime.now()
        
        response = agentcore_client.invoke(
            runtime_id=runtime_id,
            payload={"query": f"Test query {i}"},
            session_id=session_id  # SAME session = warm start
        )
        
        end_time = datetime.now()
        latency_ms = (
            end_time - start_time
        ).total_seconds() * 1000
        
        latencies.append(latency_ms)
        
        # Keep session alive
        session_manager.record_invocation(session_id)
    
    # Step 4: Calculate percentiles
    return calculate_percentiles(latencies)

def calculate_percentiles(latencies: list) -> dict:
    """Calculate statistical distribution"""
    sorted_latencies = sorted(latencies)
    n = len(sorted_latencies)
    
    return {
        'p50': sorted_latencies[int(n * 0.50)],
        'p90': sorted_latencies[int(n * 0.90)],
        'p95': sorted_latencies[int(n * 0.95)],
        'p99': sorted_latencies[int(n * 0.99)],
        'mean': sum(latencies) / n,
        'std_dev': statistics.stdev(latencies),
        'min': min(latencies),
        'max': max(latencies),
        'raw': latencies
    }
```

**4b — Why 100 Iterations:**
```
Statistical reasoning:
- p99 requires minimum 100 samples for accuracy
- p95 requires minimum 20 samples
- p50 requires minimum 10 samples

With 100 iterations:
- p99 = 1 sample at 99th percentile
- Captures occasional outliers
- Stable mean and std dev
- Reliable SLA assessment
```

```python
import boto3

def publish_performance_metrics(
    metrics_data: dict,
    runtime_id: str,
    environment: str,
    test_type: str
):
    """
    Publish performance metrics to CloudWatch
    Namespace: AgentCore/PerformanceTesting
    """
    cloudwatch = boto3.client('cloudwatch')
    
    # Common dimensions
    dimensions = [
        {'Name': 'runtime_id',   'Value': runtime_id},
        {'Name': 'environment',  'Value': environment},
        {'Name': 'test_type',    'Value': test_type},
        {'Name': 'timestamp',    
         'Value': datetime.now().isoformat()}
    ]
    
    metric_data = []
    
    if test_type == 'cold_start':
        # Publish cold start metrics
        for metric_name, value in [
            ('ColdStartTime',      
             metrics_data['total_cold_start']['mean']),
            ('ContainerInitTime',  
             metrics_data['vm_init']['mean']),
            ('RuntimeStartupTime', 
             metrics_data['runtime_startup']['mean']),
            ('FirstResponseTime',  
             metrics_data['first_request']['mean']),
        ]:
            metric_data.append({
                'MetricName': metric_name,
                'Value':      value,
                'Unit':       'Milliseconds',
                'Dimensions': dimensions
            })
    
    elif test_type == 'warm_invocation':
        # Publish warm invocation metrics
        for metric_name, value in [
            ('WarmInvocationLatency_P50', 
             metrics_data['p50']),
            ('WarmInvocationLatency_P90', 
             metrics_data['p90']),
            ('WarmInvocationLatency_P95', 
             metrics_data['p95']),
            ('WarmInvocationLatency_P99', 
             metrics_data['p99']),
            ('WarmInvocationLatency_Mean', 
             metrics_data['mean']),
            ('WarmInvocationLatency_StdDev', 
             metrics_data['std_dev']),
        ]:
            metric_data.append({
                'MetricName': metric_name,
                'Value':      value,
                'Unit':       'Milliseconds',
                'Dimensions': dimensions
            })
    
    # Publish in batches of 20 (CloudWatch limit)
    for i in range(0, len(metric_data), 20):
        batch = metric_data[i:i+20]
        cloudwatch.put_metric_data(
            Namespace='AgentCore/PerformanceTesting',
            MetricData=batch
        )
    
    print(f"Published {len(metric_data)} metrics to CloudWatch")
```

#### **Section 9: Regression Detection (~300 words)**

**What to Write:**
```python
def detect_regression(
    current_metrics: dict,
    baseline_metrics: dict,
    threshold_pct: float = 20.0
) -> list:
    """
    Detect performance regressions vs baseline.
    Triggers alert if degradation > 20%
    """
    regressions = []
    
    metrics_to_check = [
        'p50', 'p90', 'p95', 'p99', 'mean'
    ]
    
    for metric in metrics_to_check:
        current = current_metrics.get(metric, 0)
        baseline = baseline_metrics.get(metric, 0)
        
        if baseline == 0:
            continue
            
        degradation_pct = (
            (current - baseline) / baseline
        ) * 100
        
        if degradation_pct > threshold_pct:
            regressions.append({
                'metric': metric,
                'baseline_ms': baseline,
                'current_ms': current,
                'degradation_pct': degradation_pct,
                'severity': 'CRITICAL' 
                    if degradation_pct > 50 
                    else 'WARNING'
            })
    
    return regressions
```
